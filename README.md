# e-inventaris
Website Inventaris Barang Sederhana dengan fitur-fitur:
- Pencatatan detail barang.
- Pencarian dan filterisasi barang.
- Pembaruan stok secara real-time.

Untuk demo website, https://einventaris.000webhostapp.com/
